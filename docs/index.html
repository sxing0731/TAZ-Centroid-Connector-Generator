TAZ Centroid Connector Generator

A local Python GIS workflow for generating, reviewing, and exporting TAZ centroid connectors from TAZ polygons, HERE Master LINKS, GSTDM LINKS, and GSTDM Master NODES.

Sector-Based Candidate Search

Splits each TAZ around its centroid, calculates HERE link density by sector, and prioritizes well distributed candidate directions.

Non-Major Node Snapping

Derives GSTDM node MAJOR_LEVEL from connected link functional classes and blocks snaps to major intersections.

QAQC Web Review

Provides a local browser review tool for inspecting TAZ context, editing connector endpoints, adding connectors, and exporting Cube DBF records.

Run Locally

py -3.12 -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt
python main.py

For configured local inputs, run the full workflow and review map generation with:

python main.py --run-default --maps --output output

QAQC Web App

.\run_qaqc_web.ps1

Then open http://127.0.0.1:8765. The app is intentionally local because it reads GIS outputs and writes QAQC files on the workstation.

Input datasets and generated outputs are not included in the public repository.
